Loops (while)

A while loop, runs a loop while a condition is still met. For example:

while (condition) {
Code to run
}

In the Code to run there must be something that affects the conditional statement, otherwise, the elements in the conditional statement will never change. A while loop will continue infinitely if the condition is never met, this is called an infinite loop.

A good while loop:

while (numDogs > numCats) {
System.out.println(“There are still more dogs than cats.”);
numDogs = numDogs -1;
System.out.println(“A dog has escaped!”);
}
System.out.println(“There is an equal amount of dogs and cats.”);

This is a good while loop as the loop actually is completed!

A bad while loop:

while (treeAge > 2) {
treeAge = treeAge + 1;
System.out.println(“The big tree is “ + treeAge + “ years old!”);
}

While it is good to celebrate trees and their birthdays, this tree does not seem to ever die and will infinitely age, causing the tree to crash the website. This is why infinite loops are bad, because a compiler will never be able to parse through them.
