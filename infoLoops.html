Loops (for)

Loops are used to iterate through code a set amount of times, or until a statement is not valid. This is very useful and fundamental for making anything meaningful. A for loop is used to iterate through a block of code a predetermined amount of times and is written in the format of:

for (statement1; statement2: statement3) { Code to iterate }

Statement1: is executed once, prior to the loop starting.

Statement2: declares the condition for the code continue iterating for

Statement3: is executed every time the code is run

And the code to iterate is defined in the code block.

The main way to use a for loop is to iterate a variable (i) for statement1 as 0.

Write the conditional expression using a mathematical logic conditional statement to set how many times you wish for the loop to run
( for three iterations, i < 3 . Isn’t that cute).

Statement3 is used to increment the variable initialed in statement1, so that each iteration it progresses to exceed the statement 2 condition.

Example

for (int i = 0; i < 3; ++i) {
System.out.println(“Iteration no.”+i);
}

Will print:
Iteration no.0
Iteration no.1
Iteration no.2
